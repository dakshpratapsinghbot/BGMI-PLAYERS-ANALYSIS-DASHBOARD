# BGMI Player Stats Dashboard

## Overview
This Tableau dashboard project provides an interactive analysis of BGMI player statistics, performance metrics, weapon usage, and map popularity. The dashboard helps visualize player behavior, win rates, and gameplay trends using multiple charts and visualizations.

---

## Dashboard Features

### Top Players by Kills
- Displays the highest-performing players based on total kills.
- Helps identify top-ranked players in the dataset.

### Player Performance Scatter Plot
- Shows the relationship between:
  - Matches Played
  - Win Rate %
- Useful for analyzing consistency and player efficiency.

### Win Rate by Category
- Compares player win rates across different categories such as:
  - Casual
  - Competitive
  - Streamer
  - Pro
  - YouTuber
- Uses box plots for distribution analysis.

### Map Popularity
- Displays the most played/popular BGMI maps.
- Includes maps such as:
  - Sanhok
  - Erangel
  - Livik
  - Miramar

### Weapon Performance
- Analyzes weapon usage and performance statistics.
- Includes popular weapons like:
  - M762
  - AWM
  - Scar-L
  - M416
  - AKM

## Dataset Information
The dataset contains BGMI player statistics including:
- Player names
- Total kills
- Matches played
- Win rates
- Weapon usage
- Preferred maps
- Player categories

---

## Key Insights
- Players with higher matches played generally show more stable win rates.
- Sanhok is one of the most popular maps in the dataset.
- M762 is among the most used weapons.
- Competitive and streamer categories show wider performance variation.
